使用nvm关了node的版本

发表于2023-04-26|更新于2024-12-18
|阅读量:

nvm全英文也叫node.js version management，是一个nodejs的版本管理工具。nvm和n都是node.js版本管理工具，为了解决node.js各种版本存在不兼容现象可以通过它可以安装和切换不同版本的node.js

下载地址

安装完成后打开cmd，输入nvm

安装完成后打开cmd，输入nvm list available 查看可以使用的node版本

安装指定node版本 nvm install 16.20.0

查看已安装的node版本 nvm ls

使用指定node版本 nvm install 16.20.0

查看使用的node版本 node -v
