Docker 属于Linux容器的一种封装

Docker 是一个开源的商业产品，有两个版本：社区版（Community Edition，缩写为 CE）和企业版（Enterprise Edition，缩写为 EE），具体使用参照官方 https://docs.docker.com/engine/install/

docker-compose

Compose 是用于定义和运行多容器 Docker 应用程序的工具。通过 Compose，您可以使用 YML 文件来配置应用程序需要的所有服务。然后，使用一个命令，就可以从 YML 文件配置中创建并启动所有服务。

Docker 的三个重要概念

  • 1、DockerFile：主要指定有哪些程序、有些什么依赖、有些什么样的配置，之后把dockerfile交给docker编译，也就是docker build
  • 2、Image（镜像）：需要dockerfile才能生成image，dockerfile可以说是image的源代码
  • 3、容器
