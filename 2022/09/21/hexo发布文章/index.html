记录我的一个坑，hexo发布文章

发表于2022-09-21|更新于2024-12-18
|阅读量:

本来很早之前就把hexo环境全部搭建好了，一直没管，今天准备来记录一些工作中遇到的问题，突然发现不知道怎么上传文章。一直往复杂的去做了，网上找的资料，还在服务器上去弄，实际完全没必要。以下为具体步骤，记录一下

安装 Hexo
所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

进阶安装和使用
对于熟悉 npm 的进阶用户，可以仅局部安装 hexo 包。

1、把之前搭建好的源码拉取下来

2、直接把.md文件复制到source/_post文件夹中（文章编辑）

3、在根目录上打开命令窗口，执行 Hexo g 生成静态文件

4、 Hexo s 本地预览(可不需要)

5、 Hexo d 发布到服务器

我真是蠢到家了
