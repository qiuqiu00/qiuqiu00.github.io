记录有关node版本以及镜像的一些问题

发表于2022-09-23|更新于2024-12-18
|阅读量:

vue项目在运行的时候报这个错 Node Sass does not yet support your current environment: Windows 64-bit with
找了很多资料，以下这种方法也尝试了

后来才发现是node.js的版本升级太高了，之前低版本的项目运行不起来

进入nodejs以往发布的旧版本列表，找到合适版本下载，卸载现有的版本，安装下载的版本。

推荐安装后缀为msi文件，msi文件是Windows Installer的数据包，它实际上是一个数据库，包含安装一种产品所需要的信息和在很多安装情形下安装（和卸载）程序所需的指令和数据。通过安装给出的提示，用户只需要进行选择即可，各种配置文件都是已经安装好的。

  • 遇到如果不是业务上的，可以考虑版本问题
  • 其次考虑镜像的问题
  • 有可能是国外资源，需要翻墙
  • npm 不行换yarn来执行
