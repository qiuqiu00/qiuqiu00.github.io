TypeScript

  1. 以JavaScript 为基础构建的语言
  2. 扩展了JavaScript并添加了类型
  3. 可以在任何支持Javascript的平台中执行
  4. TS不能被JS解析器直接执行,需要编译为JS后执行

TypeScript 开发环境搭建

  1. 下载node.js
  2. 安装node.js
  3. 使用npm全局安装TypeScript
    • 输入 npm i -g typescript
  4. 创建一个ts文件按
  5. 使用tsc对ts文件进行编译
    • 进入命令行
    • 进入ts文件所在目录
    • 执行命令：tsc xxx.ts

    webpack 打包ts

    1. 创建一个文件夹,初始化 执行 npm init -y，会自动生成一个package.json的文件
    1. 安装开发依赖-D(save dev)等,执行命令 npm install -D webpack webpack-cli typpescript ts-loader
      package.json 会自动生成一下配置
    1. 编写webpack的配置文件，创建webpack.config.js
    1. 创建ts的编译规范配置文件，在根目录创建tsconfig.json

    1. package.json里面的script加入一个bulid的命令

      就可以进行npm run build进行打包
